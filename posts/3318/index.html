tmux常用命令

发表于2023-05-30|更新于2023-06-01|实用
|阅读量:

会话管理

创建会话

tmux new -s 会话名称

退出会话

detach 或快捷键 Ctrl+b 然后再敲c

会话列表

终端：tmux ls

切换：tmux attach（或a） -t 会话名

会话内切换：快捷键 Ctrl+b s

会话重命名

tmux rename-session 旧会话名 新会话名

快捷键：Ctrl+b $ 重命名当前会话

会话销毁

tmux kill-session -t 会话名

会话中：Ctrl+b 加冒号，再输入 tmux kill-session

窗口管理

窗口划分

上下 tmux split-window | ctrl+b “

左右 tmux split-window -h | ctrl+b %

移动光标 tmux select-pane -U （上下左右：UDLR）

关闭当前面板 tmux kill-pane

窗口新建

tmux new-window (-n 窗口名)

Ctrl+b c

窗口切换

tmux select-window -t 窗口编号或名

切换到上一个 Ctrl+b p

切换到下一个 Ctrl+b n

从列表中选择窗口 Ctrl+b w

Ctrl+b 编号

窗口重命名

tmux rename-window 新窗口名

其他

ctrl+b ? 列出所有快捷键

ctrl+b z：当前窗格全屏显示，再使用一次会变回原来大小

ctrl+b ctrl+：按箭头方向调整窗格大小

ctrl+b q：显示窗格编号

ctrl+b {：当前窗格与上一个窗格交换位置

ctrl+b }：当前窗格与下一个窗格交换位置

tmux source-file ~/.tmux.conf 重新加载当前的 Tmux 配置

删除所有拆分窗口 CTRL+b :kill-pane -a

同步窗格

ctrl+b : setw synchronize-panes（支持tap补全）

这个选项值针对某个窗口有效，不会影响别的会话和窗口。 完事儿之后再次执行命令来关闭
