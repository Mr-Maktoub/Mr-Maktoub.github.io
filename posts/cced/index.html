Stable-Baselines3 Docs - 可靠的强化学习实现

Stable Baselines3 (SB3)是 PyTorch 中强化学习算法的一组可靠实现。它是Stable Baselines的下一个主要版本。

Github 仓库：https ://github.com/DLR-RM/stable-baselines3

论文：https ://jmlr.org/papers/volume22/20-1364/20-1364.pdf

RL Baselines3 Zoo（SB3 的训练框架）：https ://github.com/DLR-RM/rl-baselines3-zoo

RL Baselines3 Zoo 提供了一组预训练代理、用于训练、评估代理、调整超参数、绘制结果和录制视频的脚本。

SB3 Contrib（实验性强化学习代码，最新算法）：https ://github.com/Stable-Baselines-Team/stable-baselines3-contrib
